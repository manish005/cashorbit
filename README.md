# 📖 Overview

**CashOrbit** is a modern personal finance management application designed to help users take complete control of their money.

Track income, expenses, investments, savings goals, and budgets through a premium dashboard with interactive analytics and AI-powered financial assistance.

Whether you're a student, working professional, freelancer, or business owner, CashOrbit provides a clean and intuitive experience for managing your finances.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/cashorbit.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

# 🔥 Firebase Deployment

Build the project

```bash
npm run build
```

Deploy to Firebase

```bash
firebase deploy
```
